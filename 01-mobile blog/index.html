Remember when I first started learning front-end development. I encountered so many articles, and was so overwhelmed by how much material I would need to learn that I didn’t even know where to start.

This guide will help you navigate learning front-end development. It provides learning resources that I found effective in the past, along with supplementary explanations.

To keep this guide digestible, I broke it up into two parts. Part one goes over developing interfaces with HTML and CSS. Part two goes over Javascript, frameworks, and design patterns. If you’re familiar with HTML and CSS you can skip to part two.
